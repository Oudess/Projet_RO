# Call Center Scheduler – Planificateur Intelligent de Quarts

Projet de Recherche Opérationnelle – Optimisation par PLNE avec Gurobi

## Fonctionnalités
- Saisie intuitive de la demande en effectif
- Coûts par quart modifiables manuellement (matin / après-midi / nuit)
- Contraintes réalistes : repos garanti, interdiction nuit → matin, charge max/agent
- Résolution optimale en moins de 0.5 seconde
- Affichage du temps de résolution et du coût total
- Visualisations pro : histogramme des charges + heatmap hebdomadaire ultra-lisible
